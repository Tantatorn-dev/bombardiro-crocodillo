# About
An action-packed arcade shooter inspired by Space Invaders! Take control of Bombardiro Crocodillo and defend the skies against the relentless brainrot horde.

This game is built using the Bevy game engine in Rust, which is relying on ECS (Entity Component System) architecture for efficient game development.

# Features

## Controls

Control in this game is simple and intuitive:

- Move: Use the arrow keys or WASD to navigate Bombardiro Crocodillo across the screen.
- Shoot: Press the spacebar to fire projectiles at incoming enemies.
